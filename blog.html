Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) is programming interface for HTML & XML web documents. It's helps to interact with web page as stucture, style, content and taking form input data

How do you select an element from the DOM?

document.getElementById('elementName')

document.getElementsByClassName('elementName')

document.getElementsByTagName('elementName')

document.querySelector('elementName') & etc

What is event delegation in the context of the DOM, and why is it usefull?

Event delegation is a technique in DOM that allows you add a event of the parent element instead of individually added event to every children elements. This technique is more helpful and useful and gives you a meaningfull program.

How do you Manipulate an element's attributes and styles using DOM?

setAttribute() method help you to set a new attributes like href, src, id from DOM

To remove an attributes use removeAttribute() method

For style a DOM element first select that element. sucs as:

document.getElementById('elementName').style.color = 'pink' or

document.querySelector('elementName').style.color = 'pink' or

document.getElementByTagName('elementName').style.color = 'pink' or

document.getElementByClassName('elementName').style.color = 'pink'
